# 스토리텔링 (Part B)

장마감(15:30) 이후 실행. 종가 반영 → 코멘터리 → 투자자 일기 생성.

## 사전 조건

- 장마감 후 실행 권장 (종가 확정 필요)
- Part A(시뮬레이션)가 먼저 완료되어 있어야 함
- 프로젝트 루트: `/Users/isang-un/Desktop/personal/investment`

## 실행 절차

### Part 시작 알림
```python
from daily_pipeline import notify
notify("📝 *Part B: 스토리텔링 시작* ({date})")
```

### Step 0: 종가 반영

`notify("📊 Step 0: 종가 반영 시작")`

```bash
cd /Users/isang-un/Desktop/personal/investment && python3 scripts/core/simulate.py {date} --close
```

종가(Close) 기준으로 market_prices, 포트폴리오 평가, 순위를 재계산하여 `daily_reports` 업데이트.
매매 정보(trades_today, rebalanced_today)는 기존 시가 체결 데이터를 그대로 유지.

`notify("✅ Step 0 완료: 종가 반영")`

### Step 1: 종가 반영된 daily_reports 읽기

Supabase에서 오늘 날짜의 `daily_reports` 데이터를 조회한다:
- `rankings`: 투자자 순위
- `market_prices`: 종목별 종가
- `investor_details`: 투자자별 상세 (총자산, 수익률, 보유종목, 거래내역)

### Step 2: 데일리 코멘터리 작성

rankings, market_prices, investor_details를 분석하여 **한국어 마켓 코멘터리** 생성 (2~4문장).

포함 내용:
- 오늘의 승자/패자
- 주요 시장 동향
- 눈에 띄는 거래

문단 구분이 필요한 곳에 줄바꿈(`\n`) 삽입.

### Step 3: 투자자 일기 작성

17명 각각의 캐릭터 어투로 2~3문장. 문장 간 줄바꿈 삽입.

| 투자자 | 어투 | 예시 표현 |
|--------|------|-----------|
| A 강돌진 | 자신감 넘치는 공격적 | "확신한다", "올인했다" |
| B 김균형 | 차분하고 분석적 | "분산 효과가 나타나고 있다" |
| C 이든든 | 보수적이고 신중한 | "급할 것 없다", "안정적으로 유지" |
| D 장반대 | 역발상적 | "모두가 팔 때 샀다", "시장이 틀렸다" |
| E 정기준 | 기계적, 무감정 | "규칙대로 균등 분배", "감정 개입 없음" |
| F 윤순환 | 섹터 전문가 | "이번 주기에는 바이오가 유망하다" |
| G 문여론 | 뉴스/여론 민감 | "기사 톤이 긍정적이었다" |
| H 박기술 | 차트 분석가 | "차트가 말해주고 있다", "RSI가 과매도 구간이다" |
| I 최배당 | 배당 투자자 | "배당이 핵심이다", "꾸준한 현금흐름이 중요하다" |
| J 한따라 | 수급 추종자 | "외국인이 사는 이유가 있다" |
| K 로로캅 | 알고리즘식, 무감정 | "데이터가 말해준다", "모델이 지시한 대로 실행" |
| L 신장모 | 실현주의자, 확고한 | "수익은 실현해야 내 돈이다", "15% 찍고 1/3 정리했다" |
| M 오판단 | 냉정한 타이머 | "지금은 쉴 때다", "이평선이 말해주고 있다" |
| N 전몰빵 | 확신의 투자자, 당당한 | "분산은 무지에 대한 방어일 뿐이다" |
| O 정익절 | 냉철한 트레이더, 기계적 | "5% 찍고 바로 정리했다", "손절은 보험료다" |
| P 정삼절 | 담백한 실험자, 일관적 | "오늘도 500으로 시작이다", "복리 욕심 안 부린다" |
| Q 정채원 | 빠르고 단호한 스캘퍼 | "거래량 터지자마자 잡았다", "+4% 찍고 바로 던졌다", "오늘은 폭증 후보 없어서 관망", "30분 안에 결판낸다" |

#### Q 정채원 일기 작성 규칙
- 데이터 소스: `transactions` 오늘 레코드(investor_id="Q") + `daily_reports.investor_details["정채원"]`
- 반드시 반영해야 할 사실:
  - 오늘 매매 횟수(매수~청산 사이클 수) / 익절·손절·강제청산(매수+30분) 결과
  - 매매가 0회면 "거래량 폭증 후보 없어서 관망" 톤으로 표현
  - 일일 수익률(전일 대비) + 누적 수익률
- 분량: 2~3문장, 줄바꿈 포함
